# rock64inschpektor
Docker image for the IOTA peer manager "inschpektor" by codepleb
https://github.com/codepleb
https://github.com/codepleb/inschpektor

## usage
start docker image with `-p 8732:8732` flags

open http://ip:8732 in your browser
